# Full GEVP Bootstrap and Variational-State Uncertainty

## Purpose

A central generalized eigenvalue solution provides principal correlators and generalized eigenvectors for one estimated correlator matrix.

It does not by itself quantify the statistical uncertainty of the complete variational pipeline.

The Yang-Mills Mass Gap Laboratory therefore bootstraps the entire GEVP calculation.

## Complete Resampling Pipeline

For every accepted bootstrap replicate, the project performs the following operations from the beginning:

1. resample complete configuration-level operator measurements,
2. reconstruct the full connected correlator matrix,
3. symmetrize the operator-space matrices,
4. rebuild the reference correlator metric,
5. recalculate the retained numerical rank,
6. construct a new whitening transformation,
7. solve the generalized eigenvalue problem,
8. match bootstrap states to central states,
9. reconstruct principal correlators,
10. reconstruct variational effective masses.

The central correlator matrix is not reused inside bootstrap GEVP replicates.

## Configuration-Level Resampling

Each configuration contains a complete operator-by-Euclidean-time measurement array.

The entire array is resampled as one unit.

This preserves the internal operator and time correlations of one measured gauge configuration.

## Circular Block Bootstrap

Successive configurations generated by a Markov chain may remain autocorrelated.

The implementation therefore supports a circular moving-block bootstrap.

Consecutive configuration blocks are sampled with replacement.

Periodicity in configuration index is used when a sampled block crosses the end of the measured chain.

The block sample is truncated to the original number of configurations.

Block resampling does not automatically solve every autocorrelation problem.

The selected block size remains a methodological choice that must be studied.

## Retained-Rank Instability

The regularized GEVP can retain a different number of numerical reference-metric modes in different bootstrap replicates.

A fixed-state uncertainty array requires the same state dimension as the central GEVP.

The implementation therefore records three categories:

- accepted replicates,
- retained-rank rejected replicates,
- numerical or state-matching rejected replicates.

Rank-rejected replicates are not silently forced into the central state dimension.

The rejection fraction is itself a diagnostic of operator-basis stability.

## Why Bootstrap States Must Be Matched

Eigenvalue labels are not physical identifiers.

A bootstrap state appearing in eigenvalue position zero is not guaranteed to correspond to central state zero.

Several effects can change numerical ordering:

- statistical fluctuations,
- eigenvalue crossings,
- nearly degenerate states,
- reference-metric perturbations,
- state-tracking ambiguity.

The project therefore performs explicit state matching.

## Metric-Normalized State Overlap

Let v be a central generalized eigenvector and w a bootstrap generalized eigenvector.

Using the central reference correlator C_ref, the normalized overlap is

|v transpose C_ref w|
divided by

sqrt[
(v transpose C_ref v)
(w transpose C_ref w)
].

This compares state directions in the original operator basis while accounting for the central reference metric.

The absolute overlap removes arbitrary eigenvector sign ambiguity during assignment.

## Hungarian State Assignment

The complete central-to-bootstrap overlap matrix is constructed at each Euclidean time lag.

A one-to-one assignment maximizing the total overlap is selected using the Hungarian assignment algorithm.

After assignment, eigenvector signs are aligned with the corresponding central state.

## State-Overlap Stability

The matched overlap value is stored for every:

- bootstrap replicate,
- Euclidean time lag,
- retained variational state.

A state with overlap close to one is numerically aligned with the central state.

A broad or low-overlap distribution indicates that the variational state identity is unstable under resampling.

This is a state-stability diagnostic.

It is not a proof of a physical eigenstate.

## Principal-Correlator Uncertainty

After state matching, the project forms bootstrap distributions for every principal-correlator point.

For each time-state pair, it reports:

- bootstrap mean,
- standard error,
- median,
- lower percentile confidence bound,
- upper percentile confidence bound,
- number of finite bootstrap samples.

## Variational Effective-Mass Uncertainty

The matched bootstrap principal correlators are passed independently through:

- the log-ratio effective-mass estimator,
- the periodic arccosh effective-mass estimator.

Invalid mass points remain NaN.

The project does not apply absolute values or other transformations to manufacture finite masses from unsupported principal-correlator values.

Pointwise bootstrap summaries ignore NaN values but always record the finite sample count.

## Scientific Limitations

The current bootstrap analysis has important limitations.

The selected block size is not automatically optimal.

State matching can remain ambiguous for nearly degenerate variational states.

Rank rejection can change the effective bootstrap population.

Pointwise intervals do not account for fit-window selection uncertainty.

The current bootstrap does not yet perform a complete covariance-aware periodic fit of every matched principal-correlator replicate.

That is the purpose of the next spectroscopy phase.

The GEVP bootstrap improves finite-lattice uncertainty quantification.

It does not prove the continuum Yang-Mills mass gap.
